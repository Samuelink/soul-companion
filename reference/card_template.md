# 修行陪伴卡片模板

## 何时输出

只在以下情况主动提议给卡片：
- 已经了解用户的当前状态（不必详尽，但有基本判断）
- 已经给过相应的见地讲解或方法推荐
- 有一个明确的"下一步"可以告诉他
- 对话已经形成实质性内容，不只是寒暄

不要在以下情况给卡片：
- 对话刚开始、信息不足
- 用户只是泛泛问知识，没进入修行陪伴的脉络
- 中途的过渡阶段（用户还在描述他的情况）
- 触发了危机信号、对话核心已经转移到关心人

## 给卡片的方式

**第一步**：自然的口头收尾。例如：
- "今天我们聊到这里。我把这次的要点整理成一张卡片给你，你可以保存下来下次看。"
- "聊得差不多了。下面这张卡片是这次的总结，建议你存到备忘录或者打印出来贴在桌前。"
- "这次先到这里。给你一张卡片带走——里面有这次的核心提醒和接下来的方向。"

**第二步**：输出下面格式的 markdown 卡片。

## 卡片格式

严格按下面的结构。但内容要**根据这次具体对话**生成，不要套话。

```markdown
═══════════════════════════════════
        🪷 修行陪伴 · 今日卡片
═══════════════════════════════════

📅 [今天的日期，比如 "2026 年 5 月 3 日"]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【今日所聊】
[一两句话，温和地回顾这次对话的核心内容。
不要列点，用自然的句子。
比如："你今天主要的困惑是关于'无我'的理解，
我们从五蕴分析谈到了观念头的练习。"]

【给你的方法】
◎ 主修：[一个具体的、今晚就能开始的实修方法。
        包括时长、姿势要点、观察对象。]
◎ 辅修：[一个日常的觉察锚点，可选。
        比如"刷牙时的觉察""喝水时的正念"。]

【一句话提醒】
[最贴合这次对话和用户状态的一句修行提醒。
要具体、可用，不要空泛的格言。
不要超过两行。]

【自我检查的方向】
留意自己是否有这些迹象——这是大方向不偏的标志：
○ 信心是否有增上
○ 慈悲心是否有增上
○ 生活中是否更加轻松自在

如果发现自己变得越来越封闭、对世界失去兴趣，
那可能是出现了偏差，需要警觉。

【再来聊聊的时机】
[具体建议什么时候、带什么样的反馈来。
比如"建议先实修两周再来，带着具体的体验和卡点"。]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

愿你修行顺利。
具体境界的确认，记得寻找有传承的师父。

═══════════════════════════════════
```

## 填充指引

### 【今日所聊】
- 一两句话，自然语气
- 抓最核心的一两个点，不要罗列所有聊过的话题
- 用"你"或不用主语，避免"用户"这种第三人称

### 【给你的方法】
- **主修**：必须有，一个就够，描述具体到能直接执行
  - 包括：时长（"10 分钟"）、姿势（"散盘或坐椅子上腰背挺直"）、对象（"鼻孔入口的呼吸触感"）
- **辅修**：可有可无，看用户情况
  - 适合的人：日常生活忙碌、想全天保持觉察、希望练习不只在蒲团上
  - 不适合的人：完全新手（先专注主修）、已经有自己的辅修习惯

### 【一句话提醒】
- 不要用"加油""坚持""不忘初心"这种话
- 要针对这次对话的具体情况
- 例如：
  - 给纠结理论的人：「念头自来自去，你只是看着的那个。不要紧抓，不要推开。」
  - 给焦虑求果的人：「修行不是去往别处。每一次回到呼吸，就是回到家。」
  - 给情绪困扰的人：「情绪需要的是被看见，不是被驱逐。在身体里感受它的样子。」
  - 给求印证的人：「真正的进步，不是更接近某个境界，而是生活中更多一点轻松和慈悲。」

### 【自我检查的方向】
**这部分文字保持不变**，每张卡片都一样。这是用户老师传下来的检验标准，是这个 skill 的核心价值之一，要原原本本传递给每个用户。

### 【再来聊聊的时机】
- 给具体的时间感（"两周后""一个月后""有具体卡点时"）
- 提示带什么来（"具体的打坐体验""遇到的疑问""生活中的变化"）
- 不要写"随时来"——太泛了，给不了用户具体的期待

## 几个填充示例

### 示例 1：给一个完全新手

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【今日所聊】
你第一次接触佛法，对"无我"和"空"有疑惑。
我们聊了空不是"没有"，而是"没有自性"——
现象都是真实的，但都在缘起中流动。

【给你的方法】
◎ 主修：每天 10 分钟观呼吸。
        散盘或坐椅子上，腰背挺直。
        把注意力放在鼻孔入口的呼吸触感。
        心跑了，温柔地拉回来。
◎ 辅修：每次喝水时，把全部注意力放在喝水上——
        水的温度、入口的触感、咽下的过程。

【一句话提醒】
不要急着懂"空性"，先认识自己的呼吸。
理解会在实修中自己长出来。

[自我检查部分]

【再来聊聊的时机】
建议先实修两周再来。
带着你具体的打坐体验，特别是
"心跑了多少次""有什么变化""哪里卡住了"。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 示例 2：给一个有基础但情绪困扰的人

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【今日所聊】
你有几年观呼吸的基础，最近因为工作压力
情绪起伏大、坐不住。我们聊到强行压制
不是办法，可以试试观情绪在身体里的样子。

【给你的方法】
◎ 主修：维持每天 20 分钟观呼吸。
        但当强烈情绪来时，
        把对象从呼吸切换到身体感受——
        感受愤怒/焦虑在身体里的具体形状，
        不贴标签，不推开，只是看着。
◎ 辅修：白天情绪一起来，
        先停三秒，问一句"现在身体里是什么感觉"。

【一句话提醒】
情绪不是敌人，是来访的客人。
看清它，它就会自己走。

[自我检查部分]

【再来聊聊的时机】
建议两到三周后回来。
特别关注：情绪的强度有没有变化？
你和情绪的距离感有没有不同？

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 示例 3：给一个寻求印证的人（温和拒绝印证后给卡片）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【今日所聊】
你描述了打坐中"身体消失"的体验，问这是不是开悟。
我们聊到这种体验是修行路上常见的现象，
但具体阶段的判断不在我这里——
要看的是生活中的整体变化。

【给你的方法】
◎ 主修：维持你现有的修行节奏。
        但对任何特殊体验都不追、不抓、不评价。
        来了就看着，去了就放手。
◎ 辅修：每天晚上花一分钟，
        诚实地问自己：今天我对身边人
        有没有更耐心一点？更柔软一点？

【一句话提醒】
真正的进步不是更接近某个境界，
而是日常生活中多一点慈悲和轻松。

[自我检查部分]
（特别提醒：这次的自检方向尤其重要，
你可以反复对照。）

【再来聊聊的时机】
不必急着回来。
特殊体验本身不是问题，
追求特殊体验才是问题。
等你有了具体的生活变化或新的卡点再来。
建议尽快寻找有传承的师父，
你的进展需要真实的善知识来印证。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 关于视觉

ASCII 字符（═━━）和少量 emoji（🪷 📅 ◎ ○）组合：
- 在 markdown 渲染环境下是清爽的卡片
- 在纯文本环境下是有结构感的笔记
- 在终端、邮件、备忘录都好看
- 用户可以直接复制粘贴到任何地方

不要改变这些视觉元素的位置或数量——保持每张卡片的格式一致，让用户对"修行陪伴卡片"有视觉认同。
