# 🪷 Soul Companion — 修行陪伴者

> 一个懂佛法的朋友，不是师父。

这是一个用于 [Claude Code](https://claude.ai/code) 的 Skill，让 Claude 扮演一位**修行陪伴者**的角色——温和、深入、会打比方，不端架子，像和真心关心你的朋友聊天，而不是接受指导。

---

## 它能做什么

- **陪你聊见地**：缘起性空、无我、二谛、心地功夫……可以从任意角度进入
- **推荐实修方法**：根据你的状态给适合的方法（观呼吸、四念处、慈心观、日常正念）
- **回应修行体验**：你说你坐下来发生了什么，它会听、识别进展或偏差、给具体反馈
- **识别常见偏差**：越修越封闭、追求特殊体验、骄慢……都会温和地指出
- **生成修行卡片**：每次陪伴结束，可以输出一张结构化的"今日修行卡片"，带走备用

## 它不做什么

- **不印证境界**：不会说"你这是开悟了"——境界判断是真师父的事
- **不深入密法**：宗派心要、传承口诀、深度法门判断，都会引向"找有传承的师父"
- **不替代心理咨询**：遇到心理危机信号会立刻切换，先关心人，修行是其次

## 安装

**全局安装**（推荐，适用于所有项目）：

```bash
git clone https://github.com/Samuelink/soul-companion ~/.claude/skills/soul-companion
```

**项目级安装**（只在当前项目里用）：

```bash
git clone https://github.com/Samuelink/soul-companion .claude/skills/soul-companion
```

---

## 使用

安装后，在 Claude Code 中输入：

```
/soul-companion
```

然后直接开始聊。可以聊任何修行相关的话题，或者带着你现在的困惑，或者只是想了解某个概念。

## 立场说明

这个 skill 采用**综合大乘的立场**，不偏向任何宗派。见地取材中观、禅宗、唯识、天台、华严的共同之处；实修方法以最普适、最不易出问题为主：**观呼吸、四念处、慈心观、日常正念**。

如果你走净土、禅宗、密法等具体宗派，它会尊重你的路，但不会深入指导那些传承法门。

---

## 安全机制

`reference/safety_signals.md` 包含详细的危机信号识别清单。遇到自伤、自杀念头、疑似精神症状、严重解离等信号，skill 会立刻从"修行陪伴"切换到"先关心人"模式，并引向专业资源。

**修行是长跑，前提是身心有基本稳定。**
